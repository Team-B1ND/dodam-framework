# Avatar

간단한 아이콘 기반 아바타 컴포넌트입니다. 
현재 구현은 숫자형 `size` prop만 지원하며, 내부에 `Person` 아이콘을 렌더링합니다.

## Usage

```tsx
import { Avatar } from "dodam-design-system/components";

export default function Example() {
  return (
    <div className="flex items-center gap-4">
      <Avatar />
      <Avatar size={32} />
    </div>
  );
}
```

---

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `size` | `number` | `24` | 아바타의 가로/세로 크기(px). 내부 아이콘은 size * 0.6으로 렌더링됩니다. |
