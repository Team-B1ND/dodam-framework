# Badges

숫자나 텍스트 등 특정 요소를 뱃지로 표시 / 강조하는 컴포넌트입니다.

---

## Dot Badge

점으로 특정 요소를 강조 가능합니다.

---

### Usage

#### 설치

```bash
pnpm add @dds-web/components
```

> `@dds-web/components` 패키지는 `@dds-web`의 `iconography`, `typography`, `colors`, `shapes`, `themes` 패키지에 의존합니다. 프로젝트에 해당 패키지들이 설치되어 있지 않다면, 함께 설치해주세요.
### 기본 사용법
```tsx
"use client";

import { colors } from "@dds-web/colors"
import { DatePicker } from "@dds-web/components";

export default function Example() {
  return (
    <Dot 
      size={12} 
      backgroundColor={colors.status.success} 
      dotCustomStyle={{ 
        zIndex: 1 
      }}
    />
  );
}
```
