# Iconography

DDS 아이콘 시스템은 **Mono**와 **Illustration** 두 가지 카테고리로 구성됩니다.

## Installation

```bash
pnpm add dodam-design-system/icons
```

## Usage

```tsx
// Mono 아이콘 (단색, 색상 커스터마이징 가능)
import { ArrowRight, Bell, Home } from "dodam-design-system/icons/mono";

// Illustration 아이콘 (컬러 일러스트)
import { Smile, Trophy, Globe } from "dodam-design-system/icons/illustration";
```

## Examples

### 기본 사용

```tsx
import { ArrowRight } from "dodam-design-system/icons/mono";
import { Smile } from "dodam-design-system/icons/illustration";

function App() {
  return (
    <div>
      <ArrowRight />
      <Smile size={32} />
    </div>
  );
}
```

<ComponentBlock>
  <ArrowRight color={colors.text.primary} />
  <Smile size={32} />
</ComponentBlock>

### 색상 커스터마이징 (Mono)

```tsx
import { Bell } from "dodam-design-system/icons/mono";
import { colors } from "dodam-design-system/colors";

function Notification() {
  return (
    <Bell 
      size={20} 
      color={colors.brand.primary} 
      pointer 
      onClick={() => alert("clicked!")} 
    />
  );
}
```

<ComponentBlock>
  <Bell size={20} color="var(--dds-color-brand-primary)" pointer />
  <Bell size={24} color="var(--dds-color-status-error)" pointer />
  <Bell size={28} color="var(--dds-color-status-success)" pointer />
</ComponentBlock>

### 버튼과 함께 사용

```tsx
import { ArrowRight } from "dodam-design-system/icons/mono";

function Button({ children }) {
  return (
    <button>
      {children}
      <ArrowRight size={16} color="currentColor" />
    </button>
  );
}
```

<ComponentBlock>
  <button className="flex items-center gap-2 px-4 py-2 bg-brand-primary text-white rounded-md">
    시작하기
    <ArrowRight size={16} color="currentColor" />
  </button>
</ComponentBlock>
