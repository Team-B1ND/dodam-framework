# Shapes

DDS의 Shape 토큰은 컴포넌트의 border-radius 값을 일관되게 관리합니다.

## Installation

```bash
pnpm add dodam-design-system/shapes
```

## Usage

### Emotion / Styled-Components

```tsx
import styled from "@emotion/styled";
import { shapes } from "dodam-design-system/shapes";

const Card = () => (
  <Div>
    Content
  </Div>
);

const Div = styled.div`
  border-radius: ${shapes.large};
`;

```

### Tailwind CSS

```css
/* globals.css */
@import "dodam-design-system/shapes/tokens.tailwind.css";
```

```tsx
// 사용 예시
<div className="rounded-medium">Content</div>
```

## Examples

### 카드 컴포넌트

```tsx
import { shapes } from "dodam-design-system/shapes";

const Card = ({ children }) => (
  <div 
    style={{ 
      borderRadius: shapes.large,
      padding: "20px",
      backgroundColor: colors.background.surface,
    }}
  >
    {children}
  </div>
);
```

<ComponentBlock>
  <div className="p-5 bg-background-surface border border-border-normal rounded-large w-full">
    <p className="text-text-primary font-medium">카드 컨텐츠</p>
    <p className="text-text-tertiary text-sm mt-1">shapes.large (18px) 적용</p>
  </div>
</ComponentBlock>

### 버튼

```tsx
import { shapes } from "dodam-design-system/shapes";

const Button = ({ children }) => (
  <button 
    style={{ 
      borderRadius: shapes.small,
      padding: "8px 16px"
    }}
  >
    {children}
  </button>
);
```
